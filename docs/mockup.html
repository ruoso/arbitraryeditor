DOCKSPACE

Any view, any container

A uniform split-tree of tabbed containers — no privileged editor area. The tool rail is home base: close anything, reopen it from Views.

TWO CANVASES

Viewport ‖ Hero

Because the canvas is a view and the viewport is a camera, you paint through Viewport while a second canvas looks through Hero and previews its 1920×1080 export live.

OVERVIEW

Patterned z-order

Render-free map: a front cell's fill occludes the one behind; dotted borders stay on top so extents read. The pattern is each layer's shared identity with the list.

BRUSH

Size in % of view

Measured against the camera, not a pixel grid — the ring is the truth; “≈48px on Retouch” is only the effect on that cell.

RESOLUTION

On cells & cameras

No document resolution. A cell emits detail; a camera samples it. The health chip flags when a camera out-resolves a cell.
